# Algoritmo Genético para TSP (Problema del Viajante)

## Instalación

```bash
# Clonar el repositorio
git clone https://github.com/S4vi0r17/TS-Genetic.git

# Navegar al directorio
cd TS-Genetic

# Instalar dependencias
npm install
```

## Uso

Para ejecutar el programa:

```bash
npm start
```
